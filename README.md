A web app for pickleball players. Built with React + TypeScript (frontend) and Node.js + Express (backend), with Firebase Authentication for Google Sign-In.

## Getting Started

### Backend
```bash
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```

### Firebase Setup
1. Create a project at [console.firebase.google.com](https://console.firebase.google.com)
2. Enable **Email/Password** and **Google** sign-in providers
3. Copy your web app config into `client/.env`
